# Privacy Policy

URL Pull collects tab titles and URLs only when the user clicks an export button in the extension popup.

## Data Handled

URL Pull can access:

- The title and URL of the current tab.
- The titles and URLs of tabs in the current browser window.
- The titles and URLs of tabs across open normal browser windows.
- Context text entered by the user in the extension popup.

## Local Storage

The context text entered by the user is saved locally with `chrome.storage.local` so it remains available after the popup or browser window is closed.

## Data Sharing

URL Pull does not send tab data, URLs, titles, saved context text, or generated exports to any server.

URL Pull does not use analytics, advertising, tracking, or third-party network services.

## User Control

Users choose when to generate an export and whether to copy or download it. Users can edit or remove the saved context text at any time from the popup.
